# Web Dev Starter Code

## Overview

A basic blog post utilizing css formatting skills to include grids, sticky navigation bar, 
and two column layout. 

## How to run:
- Open your terminal and cd into the directory you wish to clone the project into
- Use the Git Clone command to copy the repository: https://github.com/Megan-Aker/blogpost.git
- Once it is copied, cd into the repo
- Open the cloned repo in your IDE, then use live preview to view index.html to view the site.
